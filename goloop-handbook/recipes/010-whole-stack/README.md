# Recipe: 010-whole-stack

Part of the [GoLoop One handbook](../../main.md). Read the chapter:
[10-whole-stack](../../chapters/10-whole-stack.md).

A notes API that ties the earlier chapters together: config, lifecycle,
observability, logging, routing, responses, middleware, validation, auth and a
pgc-generated store.

Run it against a real database:

```sh
docker compose up -d
export PGC_DATABASE_URL='postgres://notes:secret@127.0.0.1:5441/notes?sslmode=disable'
pgc migrate
go test ./...          # skips when no database is configured
go run .               # serves on :8087; Ctrl-C for a graceful shutdown
```

`internal/store/` is generated by `pgc generate` and committed so the recipe runs.
